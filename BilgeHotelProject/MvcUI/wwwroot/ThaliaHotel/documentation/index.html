Installation

Follow the steps below to get started with your Site Template:

  • Open the Package/Thalia template Folder to find all the Templates Files
  • You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  • Make sure you upload the required files/folders listed below:
    • Thalia template/css – Extra Stylesheets Folder
    • Thalia template/img – Images Folder
    • Thalia template/js – Javacripts Folder
    • Thalia template/index.html – Index File/Homepage
  • You’re now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

Logo Settings

The Logo Container can be found in div with '.logo' class


	<div class="logo">
		<img src="img/logo.png" alt="">
	</div>

Retina Logo and other images

If you wish to use a Retina Logo, or any other image, make sure that you have double the size of your Standard image in the "img" folder.

Color Schemes

Main color code can be find in: Thalia template/css/colors/color.css

Changing Fonts

You can find the Linking to the Font Files in the head tag of all the .html files.

Template Customization

We have include a style CSS File in the css Folder so that you can better handle your Customizations while updating Thalia to the Latest Version. Simply add all your Custom CSS Codes in the css/style.css File and link it in the Document before the css/colors/color.css Linking. Also make sure that this is the Last Linked CSS File in the Document so that your Custom CSS Styles are Overwritten properly.
